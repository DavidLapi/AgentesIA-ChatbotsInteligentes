# Cómo Funciona el Web Scraper de IA

Aquí hablaremos sobre cómo funciona el raspador web de IA antes de ejecutar la aplicación.

- Primero, *el usuario ingresa una URL de un sitio web*, así que si quieres raspar un sitio web, debes proporcionar una URL.

Entonces digamos que vas a Wikipedia o alguna página o sitio de noticias, y obtienes una URL de un sitio web.

- *La IA raspará (scrapes) y extraerá el texto* de esa URL en particular.

- Luego, *la IA genera una versión resumida de la página*, así que tomará ese contenido y lo resumirá.

- Finalmente, *los resultados se muestran en nuestra interfaz web*, que está impulsada por **Streamlit UI**.

Este sería todo el proceso. Sólo tiene que encontrar la URL que deseas usar y luego dársela al sitio web (al raspador), y la IA hará todo el trabajo de *rasparla*, *resumirla* y *darte el resultado*.
