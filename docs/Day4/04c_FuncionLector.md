# Cómo Funciona el Lector de Documentos de IA

Aquí vamos a hablar de cómo funciona el lector de documentos de IA.

1. Aquí el usuario primero sube un **documento PDF** a través de esa interfaz de usuario de Streamlit que hemos proporcionado. El usuario irá allí. 

2. Encontrará un botón para hacer click y subir un documento PDF. Cuando el usuario esté navegando en su computadores, sólo le mostrará el documento PDF.

3. Luego la IA *extrae y almacena* el texto en archivos. Para cualquier documento que se haya subido, tomará el texto de ese documento y lo guardará en archivos.

4. Desupués el usuario hará una pregunta sobre el documento. Así que en nuestro caso, vamos a usar uno de mis *CVs antiguos* que tengo disponible para subirlo, y luego tú puedes preguntar sobre mí a través de esta IA de respuesta a preguntas.

5. Finalmente, la IA recupera la información más relevante y responde la pregunta por nosotros.

Así que eso es todo lo que hace este proyecto. Entonces vamos a *proceder y ejecutar* el lector de documentos de IA.
