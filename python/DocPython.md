## Preparar nuestro espacio de programación

1. Instalaremos Python

Accedemos a https://www.python.org/ y nos mostrará el sitio web donde puedes aprender todo sobre Python. Ve a **Download** y asegúrate de qué computadora estás usando Windows, Linux, MacOS u otro sistema perativo a instalar la útlima versión de Python.

2. ¿Por qué estamos haciendo esto?

- Necesitas Python en tu computadora para que entienda el código mágico que escribirás.
- Piénsalo como instalar un traductor que habla tanto computadora como inglés.
