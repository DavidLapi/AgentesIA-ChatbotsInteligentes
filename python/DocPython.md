## Primera línea de código

Lo que vamos a escribir aquí es escribir nuestro primer script en Python.

Para hacerlo nos vamos a nuestro espacio de programación (en nuestro caso, VS Code) y nos dirigimos a nuestra ruta del proyecto:

```
C:/Users/alumno/Desktop/AIAgents$
```

En el menú de inicio superior izquierda de VS Code, haz click en Archivo --> Nuevo archivo... Después nombra el archivo "primer_script.py". 

Éste será nuestro primer archivo en el que empezaremos a programar, definiendo que "primer_script" es el nombre del archivo (se puede renombrar) y ".py" es el prefijo que define un archivo Python.

## Funciones

Las funciones en Python son tareas para que la realice una computadora.

En el archivo creado, escribiremos:

```py
print("Hola mundo!")
```

"print" es una función que devuelve un valor, ya sea una cadena de texto (string), un número (integer) u otro tipo de dato (float, bool, dict...)

Para ejecutar el archivo, abre el terminal de la computadora. Selecciona la ruta donde se encuentra el archivo Python y ejecuta el siguiente comando:

```bash
python primer_script.py
# Resultado: Hola mundo!
```
